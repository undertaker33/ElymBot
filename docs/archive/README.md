# 归档

完成、失效或不再活跃的历史材料放在这里。归档是历史证据，不是当前事实来源。

## 子目录

- `pre-uth-docs/docs-00-11/`：接管前旧 `docs/00_*.md` 到 `docs/11_*.md`，已由编号 context 取代。
- `takeover-repair/`：本轮接管修复早期 seed 与失效完成证据。
- `context-pre-numbering/`：旧未编号 context 迁移证据。
- `work/`：正式任务包归档。
- `LW-Work/`：轻量任务记录归档。

归档文件中的旧提交号、旧路径、旧版本号和旧完成态不得覆盖 `docs/current-state.md` 与 `docs/context/` 的当前事实。
