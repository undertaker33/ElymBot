# 治理规则

项目治理规则放在这里。场景路由和场景执行由已安装的 `uth-*` 技能负责，不在本目录重复实现。

## 文件

- `agent-rules.md`：稳定 agent 行为规则。
- `git-workflow.md`：Git 写入与发布规则。
- `subagent-workflow.md`：worker / planner / evaluator 协作规则。
- `writing-rules.md`：Design / Todo / Feedback / LW / context 写入规则。
- `hook-gates.md`：L0 / L1 / L2 / L3 gate。
- `state-rules.md`：current-state 与 snapshot 规则。
- `adr-release-rules.md`：ADR 与 changelog 边界。
