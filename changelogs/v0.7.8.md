# v0.7.8

## 新增

- 新增 `HiltLlmProviderProbePort`，将 LLM Provider 探测正式纳入 Hilt 生产图。
- 新增多组 legacy repository adapter 的兼容测试覆盖，确保 feature-first 迁移后旧入口仍可被约束。
- 新增架构合同对 LLM source、feature 边界和 root 边界的校验。

## 修复

- 修复 legacy adapter 与新 repository port 之间的 selected state、Cron 调度和资源中心兼容问题。
- 修复 `PluginRuntimeFacade` 与 LLM 探测适配器中仍保留的旧运行时耦合。
- 修复架构测试对新 feature-first 路径识别不完整的问题。
