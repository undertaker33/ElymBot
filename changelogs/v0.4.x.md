# v0.4.x

## 新增

- 新增插件仓库、插件目录、插件目录同步、插件安装意图处理与仓库订阅管理。
- 新增插件详情页、插件配置页、插件工作区、插件运行日志页与插件触发管理入口。
- 新增插件静态配置 schema、配置快照、配置边界与 schema 渲染能力。
- 新增外部插件执行契约、工作区策略、执行策略、治理模型与发布校验。
- 新增插件执行宿主能力、宿主动作执行、执行结果合并、失败分类与运行时调度能力。
- 新增 QuickJS 插件治理与工具链基础，推动插件能力从 Python 方案迁移到 QuickJS 方案。
- 新增插件开放触发器：插件入口点击、命令、发送前、模型回复后。
- 新增插件工作区资源标识约定。
- 新增日志定时清理控制与插件页面统一操作按钮。
- 新增下载进度弹窗、下载任务数据库、前台下载服务、下载通知与可恢复下载器。
- 新增插件包契约快照、包契约解析与安装校验问题模型。
- 新增插件样例、安装升级样例与插件相关说明文档。

## 修复

- 修复插件检测卡片不可用问题。
- 修复下载通知权限缺失导致通知不可用的问题。
- 优化插件页操作按钮、确认弹窗、详情信息、运行日志和配置页展示。
- 优化 OneBot 载荷解析、插件安装器、插件目录同步与插件仓库数据一致性。
- 优化数据库插件表结构和迁移，补齐插件目录、配置快照、下载任务、包契约等 schema。
- 优化插件与聊天链路、OneBot 桥接和宿主动作执行之间的集成稳定性。
