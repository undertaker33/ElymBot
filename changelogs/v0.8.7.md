# v0.8.7

## 新增

- 新增 `app-integration` 模块，集中承接 Bot、Config、Cron、Persona、Provider、Resource 等仓库绑定。
- 新增 Profile 引用检查绑定和运行时日志绑定。
- 扩展 GitHub Actions CI，覆盖更多 Gradle、架构合同和模块边界检查。

## 修复

- 修复 profile 删除保护、QQ 数据映射、repository port 和运行时日志 owner 分散的问题。
- 修复 `app` 直接承担过多 repository port 绑定的问题，将集成责任迁入 `app-integration`。
- 修复 CI smoke workflow 覆盖不足的问题。
