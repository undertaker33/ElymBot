# v0.5.x

## 新增

- 新增插件 v2 QuickJS 运行时基础，包括启动、注册表、运行时会话、会话工厂、分发信封与分发引擎。
- 新增插件 v2 主动运行状态存储、宿主启动接口和运行时加载器。
- 新增插件 v2 命令解析、消息合约、生命周期合约、过滤器、生命周期管理与宿主入口处理。
- 新增插件 v2 LLM 合约、模型请求前后处理、事件结果协调和 LLM 流水线协调器。
- 新增插件执行结果合并、运行时日志总线、外部脚本执行与外部插件集成测试路径。
- 新增插件 v2 回调生命周期、外部表情插件、消息插件、LLM 插件等 QuickJS 集成测试资源。
- 新增按模块拆分后的界面包结构，将应用、导航、聊天、配置、插件、设置、语音资产等页面拆入更清晰的目录。
- 新增插件 schema 按钮布局策略与插件管理展示测试。

## 修复

- 修复插件运行时日志、脚本执行、启动接口、生命周期回调和消息处理中的多处边界问题。
- 修复流式响应分段相关问题，并补充专门测试。
- 优化插件目录同步、静态配置解析、OneBot 桥接、ChatViewModel 与 PluginViewModel 的协作。
- 优化页面组织和导航结构，降低顶部导航与页面实现的耦合。
- 优化插件 v2 从一期收口到 QuickJS 执行链路的稳定性，为后续正式整合做准备。
