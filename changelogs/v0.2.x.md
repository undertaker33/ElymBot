# v0.2.x

## 新增

- 新增 Bot 绑定 QQ 能力，绑定后对应账号可默认使用该 Bot 回复。
- 新增 QQ 聊天页 `/reset`、`/persona` 指令。
- 新增对话数据本地持久化开关。
- 新增配置模块，支持多配置编辑、现实世界时间感知与后续语音/图片能力占位。
- 新增深色模式，支持白天、夜晚、跟随系统。
- 新增中英文资源，支持应用内双语切换。
- 重做模型页，按模型能力标签组织模型。
- 新增 Gemini、Ollama、Qwen、智谱、xAI 等对话模型提供商支持。
- 新增图像理解检测与图片转述链路，非多模态模型也可通过转述回复图片内容。
- 新增 QQ 对话展示开关，聊天页可选择是否展示 QQ 对话。
- 新增流式输出分段、语音风格映射、TTS 音色目录等语音能力基础。
- 新增端侧语音相关能力，包括 Sherpa ONNX 接入、端侧 TTS 资产、音色资产管理与声音克隆服务。
- 新增缓存清理与语音资产页面，支持参考音库、导入、概览、克隆音色等操作。

## 修复

- 修复默认模型绑定语义不清的问题，统一为配置页默认模型，Bot 页作为配置投影。
- 修复 Qwen3 声音克隆请求中出现的 400 与 403 问题。
- 优化部分页面、卡片和配置项显示效果。
- 清理无用资产，降低早期包体与页面噪声。
