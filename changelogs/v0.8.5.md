# v0.8.5

## 新增

- 新增 Room schema `22`，支持定时任务会话上下文和主动能力配置的持久化。
- 新增 Web Search Prompt 指引、搜索提示字符串和运行时上下文扩展。
- 新增 Cron 主动能力意图拦截、兜底回复和未来任务工具入口。
- 新增多处本地化字符串资源，覆盖 Cron、运行时提示和 Web Search 文案。

## 修复

- 修复定时任务触发时工具源和主动能力暴露过宽的问题。
- 修复 Prompt 组装、运行时上下文和定时任务会话配置之间的数据遗漏。
- 修复备份、配置映射、搜索提示和资源投影在 schema 升级后的兼容问题。
