# v0.8.x

## 新增

- 完成主要仓库与运行时边界的严格 Hilt-only 收口。
- 新增插件配置快照、插件状态存储、插件配置解析、状态 Store、状态值编码和插件数据清理。
- 新增插件配置、治理、管理、市场、工作区等展示层绑定拆分。
- 新增 Cron 立即运行端口、调度对账、定时消息投递端口和定时任务执行链路。
- 新增主动能力自然语言解析和未来任务工具，包括暂停、恢复、查看运行、更新和立即执行。
- 新增核心 Unified Search 运行时，包括搜索解析、触发规则、Provider 合约、统一协调器和 Provider 配置。
- 新增 Tavily、Brave、BoCha、Baidu AI 等搜索 Provider。
- 新增本地 Meta Search 兜底体系，包括搜索策略、相关性排序、结果合并、轻量抓取、内容裁剪、URL 安全和多个搜索引擎适配。
- 新增 QQ 新闻回复格式化和搜索结果直出链路。
- 新增 Room schema `21` 与 `22`，支持插件状态、插件配置和定时任务会话上下文配置。
- 新增定时任务意图拦截、兜底回复、主动能力提示文案和 Web Search Prompt 指引。
- 新增 Cron、运行时提示、Web Search 等本地化字符串资源。

## 修复

- 修复旧插件触发链路残留问题，统一收口到新的运行时和宿主能力合约。
- 修复插件工具 JSON 值类型不稳定的问题，增加工具值归一化。
- 修复 Cron 立即运行、调度对账、定时消息投递和任务上下文组装中的边界问题。
- 修复 Web Search 从插件私有策略迁移到核心运行时后的一致性问题。
- 修复 HTML 搜索兜底链路不稳定问题，改为本地 Meta Search 兜底。
- 修复 QQ 新闻搜索结果格式化、流式回复和直出投递中的一致性问题。
- 修复定时任务触发时主动能力工具误暴露的问题。
- 修复备份、配置映射、Prompt 组装、运行时上下文、资源中心、启动链和插件供应链路中的多处兼容问题。
- 补充插件配置状态、Cron、主动能力、Unified Search、本地搜索、QQ 新闻、Prompt、DB schema、GlobalTopBar 与 Hilt 单例白名单测试。
