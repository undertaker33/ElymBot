# v0.9.1

## 新增

- 新增 build-logic 二进制 Kotlin JVM convention plugin，保留 `elymbot.kotlin.jvm` 插件 ID。
- 新增 chat runtime 的 bot command 字符串资源归属，覆盖默认和中文资源。
- 新增 QQ runtime 对 bot command 资源解析的连接入口。

## 修复

- 修复 build-logic 预编译脚本在当前构建环境下解析不稳定的问题。
- 修复 bot command 字符串仍散落在 app 与 voiceasset 资源中的 owner 错位。
- 修复 QQ message runtime 和 runtime graph node factory 对 bot command 处理入口不完整的问题。
- 修复 CI 和架构测试中 chat runtime、QQ impl 空 source root、HTTP fixture logger 与 Cron 时区基线不一致的问题。
