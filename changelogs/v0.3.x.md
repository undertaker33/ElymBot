# v0.3.x

## 新增

- 新增 STT 提供商：Whisper、Xinference、阿里云百炼、Sherpa ONNX。
- 新增 TTS 提供商：OpenAI、阿里云百炼、MiniMax、Sherpa ONNX。
- 新增声音克隆、端侧语音识别、端侧语音回复、TTS 音色选择与参考音库管理。
- 新增 Bot 语音消息识别与语音消息发送能力。
- 新增流式文本和语音输出分段能力。
- 新增会话备份、会话持久化、会话导入导出与会话标题同步。
- 新增应用级导航、页面骨架、运行时浮层、二级页面布局与配置页目录树。
- 新增配置页管理员、会话策略、唤醒触发、回复行为、白名单、忽略权限、速率限制、关键词检测等模块。
- 新增 QQ 会话标识、QQ 群聊和私聊会话区分、QQ 回复策略、唤醒词、关键词、白名单、限速与标题解析链路。
- 新增应用级完整备份模型，覆盖 Bot、模型、人格、配置、TTS 音色等模块。
- 新增旧数据迁移、旧运行时状态导入、登录诊断、安全日志、HTTP 客户端和运行时 secret 管理。
- 新增 Bot 指令体系、主页面滑动导航、Bot 工作区、插件页雏形。
- 新增插件安装记录、插件包校验、插件 schema 渲染、插件执行协议与 App Chat 插件运行时雏形。

## 修复

- 修复 Bot 默认绑定不生效的问题。
- 修复流式响应中真正的空值被误读为字符串的问题。
- 修复 QQ 二维码频繁轮询问题。
- 修复检测卡片不可用问题。
- 修复顶部导航栏耦合过高和页面闪烁问题。
- 优化 OneBot 桥接、NapCat 登录、本地运行时安装与运行态诊断稳定性。
- 优化聊天页侧边栏、会话删除、私聊重命名、置顶与页面切换动画。
- 优化 QQ 回复格式化、会话键生成、回复策略和会话标题解析一致性。
- 重构本地数据模型与数据库结构，降低配置、会话、备份、资产管理等模块耦合。
- 补充大量数据迁移、登录诊断、HTTP、运行时、QQ 策略、备份、ViewModel 与插件基础测试。
