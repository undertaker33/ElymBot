# v0.9.3

## Changed

- Upgraded the Android compile toolchain to AGP 9.2.0 and Gradle 9.4.1.
- Switched Android modules to the AGP 9 built-in Kotlin flow and removed direct `org.jetbrains.kotlin.android` plugin usage.
- Updated compiler and annotation-processing dependencies, including KSP 2.3.8, Room 2.8.4, and Hilt 2.59.2.

## Fixed

- Resolved Gradle 9 and Kotlin 2.2 migration warnings in build logic, source set configuration, and affected unit test helpers.
- Preserved warning-free architecture, unit test, and debug APK build gates after the toolchain upgrade.
