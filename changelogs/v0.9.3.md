# v0.9.3

## 新增

- 升级 Android 编译链到 AGP `9.2.0` 和 Gradle `9.4.1`。
- Android 模块改用 AGP 9 built-in Kotlin 路线，并移除直接使用 `org.jetbrains.kotlin.android` 插件的旧写法。
- 同步升级编译器和注解处理依赖，包括 KSP `2.3.8`、Room `2.8.4` 和 Hilt `2.59.2`。

## 修复

- 修复 Gradle 9 与 Kotlin 2.2 迁移过程中 build logic、source set 配置和相关单元测试 helper 的告警。
- 保持编译链升级后的架构检查、单元测试和 debug APK 构建 gate 无告警通过。
