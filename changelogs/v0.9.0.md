# v0.9.0

## 新增

- 新增插件数据装配工厂，集中处理插件数据、配置、状态和运行时模块的 Hilt wiring。
- 新增 runtime logging、runtime audio、runtime secret、runtime network 和 runtime container 的进一步模块化收口。
- 新增数据库 schema、runtime compatibility facade、feature repository facade 和 app shell stateless 等合同测试。

## 修复

- 修复插件数据、插件运行时、Cron runtime 和运行时日志之间的 Hilt 绑定分散问题。
- 修复 Chat、LLM、音频、容器、secret、搜索和备份在拆分后仍保留旧兼容入口的问题。
- 修复全局单例、静态仓库访问、QQ runtime、runtime audio/context/container 等架构基线不一致问题。
