# v0.8.6

## 新增

- 新增 Pull Request 模板和 Android CI workflow 基础入口。
- 新增多个 presentation 级 Hilt 绑定模块，将 QQ、运行时资产和插件界面依赖从通用 ViewModel 模块中拆出。
- 新增 Global Singleton allowlist、Hilt-only 和 build baseline 的架构测试覆盖。

## 修复

- 修复 `RuntimeServicesModule` 和 `ViewModelDependencyModule` 继续承担过多展示层绑定的问题。
- 修复下载、容器、LLM、secret、备份和插件供应链路中的旧 runtime service 引用。
- 修复 CI Android SDK 配置和 Node Android SDK setup 版本问题。
