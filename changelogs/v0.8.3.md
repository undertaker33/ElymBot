# v0.8.3

## 新增

- 新增 Unified Search 合约、协调器、自然语言解析和触发规则。
- 新增 Tavily、Brave、BoCha、Baidu AI 等搜索 Provider。
- 新增 HTML 搜索兜底、结果模型、解析策略、相关性排序和 Provider 配置。
- 新增 Web Search Prompt 与运行时上下文对统一搜索的支持。

## 修复

- 修复 Web Search 仍依赖插件私有策略的问题，改为核心运行时统一协调。
- 修复搜索 Provider 配置、触发规则和 Prompt 组装之间的语义不一致。
- 修复 QQ、Cron 和 Chat 使用搜索结果时缺少统一格式化入口的问题。
