# v0.8.10

## 新增

- 新增 feature-first 物理模块拆分，覆盖 Bot、Chat、Config、Conversation、Cron、Plugin、Provider、QQ、Resource、Download 和 core UI。
- 新增 Chat runtime bot command、Cron runtime、Download API/impl、Provider API、QQ API 和 Resource presentation 等独立入口。
- 新增 App shell thin boundary、module dependency graph、feature import 和多组架构合同测试。

## 修复

- 修复 `app` 直接拥有过多 feature data、runtime 和 presentation 代码的问题。
- 修复 Bot command 字符串、Provider runtime port、Scheduled message delivery 和 runtime asset owner 的归属问题。
- 修复 feature import allowlist、static repository usage 和 global singleton allowlist 的基线不准问题。
